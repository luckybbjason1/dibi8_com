## Minara가 정확히 무엇인가

마케팅 카피를 벗기면 Minara는 **하나의 채팅 인터페이스에서 네 가지를 동시에** 합니다:

- 어떤 시장이든 **자연어로 질문**
- 실시간 호가창, 온체인 데이터, 뉴스 기반의 **실시간 분석 반환**
- 같은 프롬프트로 암호/주식/원자재 **거래 실행**
- 대화형 포지션 사이징과 진출 조건으로 **리스크 관리**

ChatGPT에 증권 라이선스와 Hyperliquid 지갑이 장착된 것이라 생각하면 됩니다.

"AI 트레이딩 플랫폼" 카테고리는 이미 붐빕니다 — TradeGPT, Numerai, Composer, Stoic. 하지만 Minara는 두 가지 면에서 다릅니다:

1. **대시보드 우선이 아닌 채팅 우선.** 모든 작업이 자연어로 진행됩니다. 차트 설정이나 JSON 전략을 쓸 필요가 없습니다.
2. **Hyperliquid 위에서 동작.** 즉, 실행이 온체인이고 투명하며 1초 이하의 최종성을 가집니다.

DeFi가 처음이면 두 번째 포인트가 와닿지 않을 수 있습니다. 계속 읽어보세요.

## Hyperliquid 백본이 왜 중요한가

세상에서 가장 똑똑한 AI를 형편없는 실행 레이어에 연결하면 슬리피지, 다운타임, 숨겨진 수수료로 돈을 잃습니다. Hyperliquid은 Minara의 AI를 실제로 가치 있게 만드는 부분입니다.

DeFi를 모르는 독자를 위한 30초 요약:

| 항목 | 중앙화 거래소(Binance, OKX) | Hyperliquid |
|---|---|---|
| 자산 보관 | 거래소가 보관 | 본인이 직접 보관 |
| 오더북 | 숨김, 오프체인 | 완전 온체인, 공개 |
| 출금 | 일시 중지/동결 가능 | 무허가, 항상 가능 |
| 일일 거래량 (2026 Q1) | 약 $300억 (Binance) | 약 $15억 (Hyperliquid) |
| 지연 시간 | 약 50ms | 약 200ms |
| 다운타임 리스크 | 점검 윈도우 있음 | 검증인 레벨 (지금까지 0회) |

트레이드오프: 약간 높은 지연을 무신뢰 실행과 카운터파티 리스크 제로로 교환. 분~주 단위 포지션 트레이드(AI 시그널이 동작하는 영역)에는 지연 차이가 보이지 않습니다. 카운터파티 차이는 매우 큽니다.

Hyperliquid은 2026년 현재 **하루 $10억+ 거래량을 전부 온체인**으로 처리합니다 — 장난감이 아닙니다.

## 3분 만에 첫 거래까지

실제 흐름:

### Step 1: Minara 계정 생성

[minara.ai](https://minara.ai/r/OSXG4X)로 이동 (이 링크는 첫날 수수료 10% 보너스를 주고, 저에게는 10% 수수료 + 20% Spark 리베이트를 줍니다 — 솔직히 공개합니다).

이메일 + 비밀번호. 기본 등급은 KYC 없음. 법정화폐 입금은 더 상위 등급.

### Step 2: Hyperliquid 지갑 연결 또는 생성

```bash
# 이미 Hyperliquid 지갑이 있다면
1. Minara 설정에서 "Connect Wallet" 클릭
2. MetaMask / Rabby / Phantom에서 서명
3. 잔액이 채팅에 자동 표시됨

# 지갑이 없다면
1. "Create Hyperliquid Wallet" 클릭
2. Minara가 새 EOA 생성 (개인키는 본인 소유)
3. 앱 내 브리지로 USDC를 어느 체인에서든 가져옴
```

지갑은 **비수탁(non-custodial)**. Minara는 자금을 절대 보관하지 않으며, 모든 거래는 본인 지갑에서 서명된 트랜잭션.

### Step 3: 첫 질문

채팅창을 열고 입력:

```
HYPE의 현재 오픈 인터레스트는? 지난주 대비 어떤가?
```

숫자와 작은 차트가 포함된 답변 + 거래 제안이 나옵니다.

### Step 4: 첫 거래

```
$200 USDC로 HYPE 영구 롱, 3배 레버리지, 5% 손절
```

Minara가 파라미터 확인 → 예상 청산가 표시 → "실행?" 질문. 확인 클릭 → Hyperliquid에서 서명 → 체결.

가입부터 첫 거래까지: **약 3분** (지갑과 USDC가 준비된 경우).

## 핵심 기능 심층 분석

### 자연어 거래 실행

이게 헤드라인 기능이자 실제로 행동을 바꾸는 기능입니다. 기존:

> Hyperliquid 열기 → HYPE 검색 → 영구 클릭 → 레버리지 설정 → 포지션 계산 → 손절 설정 → 익절 설정 → 검토 → 확인

이제는 한 문장:

```
HYPE 롱 3배, $500 포지션, 손절 40, 익절 55
```

끝. Minara가 의도 파싱 → 계약 수 계산 → 조건부 주문 설정 → 1줄 요약 → 실행. 6번 클릭이 한 문장이 됩니다.

### 실시간 AI 분석

AI는 2024년 학습 데이터가 아니라 실시간 데이터에 기반한 분석을 제공합니다. 질문:

```
BTC가 지난 10분 동안 왜 3% 급등했는가?
```

답변: "Coinbase에서 지난 8분간 $4000만 순매수, Hyperliquid 오픈 인터레스트가 12% 증가, Bloomberg가 [실제 헤드라인]에 대한 뉴스를 방금 발행. 뉴스 기반이라기보다 모멘텀 기반으로 보임."

이 부분이 모델 품질이 드러나는 곳입니다. 테스트 결과 Minara는 TradingView Premium + Bloomberg Terminal 구독에 데이터 소싱 측면에서 비등하며, 종합 속도는 더 빠릅니다.

### 크로스 자산 (암호 / 주식 / 원자재)

채팅창은 HYPE 영구든, AAPL 옵션이든, 금 선물이든 신경 쓰지 않습니다. 물어보면 거래 가능한 것을 알려주고 거래합니다.

```
앞으로 2주간 Nvidia를 롱 잡는 가장 저렴한 방법은?
```

Minara가 비교: NVDA 현물, NVDA 콜 옵션, 레버리지 ETF (NVDL), 암호화폐 상관 대체 자산. 계좌 규모에 따른 최적 리스크/리워드를 알려줍니다.

참고: 주식과 원자재는 파트너 브로커를 통해 실행, Hyperliquid 아님. Hyperliquid은 암호 영구 베뉴. 채팅 레이어는 통합, 실행 베뉴는 분산.

### Spark 토큰 경제

모든 거래는 Spark 토큰(플랫폼의 보상 토큰)을 적립. Spark는:

- 수수료 할인에 사용 (최대 50% off)
- 스테이킹으로 추가 할인 + 거버넌스
- 시장에서 판매 가능 (시장가 있음)

추천 링크로 가입하면 **양쪽 모두 Spark 보너스**: 신규 사용자는 가입 부스트, 추천인은 신규 사용자 Spark 수익의 20% (일정 기간).

이게 10% 수수료에 더해지는 "20% Spark 리베이트"입니다.

## 5가지 실전 사용 사례 (그리고 추천하지 않는 1가지)

### 사용 사례 1: "계산 없이 가설 검증"

Twitter에서 "구리/금 비율이 10년 극단치 도달"을 봤습니다. 직접 스프레드시트하기 싫습니다.

```
지금 구리/금 비율은? 역사적으로 어디인가?
이 수준에서 30일 가격 행동은?
```

Minara가 데이터 추출 → 차트 → 페어 트레이드(구리 롱/금 숏) 제안 + 사이징.

### 사용 사례 2: "Telegram bot 스타일 거래를 온체인으로"

Maestro, BananaGun, Trojan 같은 봇을 써봤다면 흐름을 압니다: 토큰 보고 → 버튼 클릭 → 체결. Minara는 같은 일을 Hyperliquid에서 (온체인, 적절한 리밋 오더, 허니팟 없음):

```
$TOKEN_TICKER를 100 USDC 시장가 매수
```

끝. 후속으로 "손절을 손익분기로 올려" 또는 "더블되면 30% 익절" 같은 명령 가능.

### 사용 사례 3: "롱 자신 있지만 하방 헷지"

```
$5K HYPE 롱 중. 한 달간 50% 하방 헷지하는 가장 저렴한 방법은?
```

Minara가 옵션 유동성 있는 상관 자산의 OTM 풋 + USDC 비용 + 손익분기 시나리오 제시 → 실행 제안.

### 사용 사례 4: "내 포지션에 영향 주는 뉴스는?"

ETH 롱 중. ETH가 갑자기 4% 떨어집니다. 질문:

```
ETH 왜 떨어졌어?
```

Minara가 동시에 뉴스, Twitter 감성, 온체인 자금흐름, ETF 인/아웃플로우, Hyperliquid 지난 1시간 청산을 확인. 단일 고래 때문인지, 뉴스 이벤트인지, 단순 디레버리지인지를 한 문단으로. 추가할지 평소가할지 결정.

### 사용 사례 5: "일일 포트폴리오 리뷰"

매일 오전 9시:

```
밤사이 P&L, 포지션 움직인 매크로 이벤트, 미리 설정한 리스크 임계치 도달 포지션 보여줘.
```

한 화면 요약. 예전에는 분석가 30분 작업.

### 추천하지 않는 사례: 스캘핑

전략이 밀리초 단위 실행에 의존(HFT 스캘핑, MEV 차익)한다면 Minara는 부적합. 채팅 인터페이스는 의도와 실행 사이에 1-3초 LLM 추론 지연. 분~주 단위 포지션엔 안 보이지만, 스캘핑엔 치명적.

그런 용도는 Hyperliquid API 클라이언트 직접 사용.

Minara는 위 모두를 완전히 대체하려 하지 않습니다. **분석 + 결정 + 실행** 루프를 압축하는 데 집중하며, 리테일 트레이더가 실제로 하는 거래(포지션 트레이드, 헷지, 기회성 베팅)에 최적화.

## 가격과 Spark 토큰 경제

무료 등급 있음 (약 50 채팅 메시지/일, 고급 기능 없음). 유료 등급은 Spotify 구독 정도부터 시작, 엔터프라이즈 "메인 트레이딩 데스크" 수준까지.

비용을 영리하게 생각하는 법:

1. **실행에서 추가 비용 없음**: Minara는 Hyperliquid 수수료에 마크업하지 않음
2. **거래로 적립한 Spark가 구독료 상쇄**: 헤비 유저는 사실상 무료 거래
3. **추천 리베이트 복리**: 친구를 데려오면 그들의 Spark 수익의 20%를 영구적으로 받음

3. **AI가 해석한 뉴스만으로 거래하지 마라.** 모델은 가끔 뉴스 맥락을 헛소리. 사이징 전 헤드라인 직접 검증.

5. **암호 영구 + AI = 레버리지 위에 레버리지.** 본인이 잃을 수 있는 금액에 대해 솔직하게.

이건 일확천금 도구가 아닌 생산성 도구. 생산성 향상은 진짜. 시장은 여전히 당신 신경 안 씁니다.
